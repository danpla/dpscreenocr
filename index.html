dpScreenOCR is a program to recognize text on screen.

Powered by Tesseract, dpScreenOCR can recognize more than 100 languages and split independent text blocks, e.g. columns. It's free and open source software that works on Windows and Unix-like systems with X11. Read the manual for information on how to install, set up, and use the program.

Download version 1.0.2 (changelog):

  • Windows: installer or ZIP archive
  • GNU/Linux
    • Debian
    • Ubuntu and derivatives
  • Source code (tar.gz)

You can contribute by translating the program into your language or by participating in development.
