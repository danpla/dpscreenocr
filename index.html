dpScreenOCR is a program to recognize text on screen.

Powered by Tesseract OCR engine, dpScreenOCR can recognize more than 100 languages and split independent text blocks, e.g. columns. It's free and open source software that works on GNU/Linux and any other Unix-like system with X11.

Read the manual for more information on how to install, set up, and use the program.

Downloads (version 1.0.0):

  • GNU/Linux
    • Debian
    • Ubuntu and derivatives

      Run in terminal:

      sudo add-apt-repository ppa:daniel.p/dpscreenocr
sudo apt-get update
sudo apt-get install dpscreenocr

      Alternatively, you can browse the PPA to download the DEB package manually.
